a11y-checker

real-time WCAG 2.1 AA accessibility auditor

axe-core playwright WCAG 2.1 AA

Quick Start

$ git clone https://github.com/zmuhls/a11y-checker.git
$ cd a11y-checker
$ npm install
$ node server.js https://example.com

Then open http://localhost:3000 to watch the audit in real time.

How It Works

Point it at any URL. The crawler automatically discovers pages via sitemap.xml, robots.txt, and link traversal. Every page is tested at desktop and mobile viewports against WCAG 2.1 Level AA criteria.

Violations stream to a browser dashboard as they're found. When the audit finishes, export a full JSON report.

Page Discovery

The crawler uses multiple strategies to find pages on your site:

  • Parses robots.txt for sitemap directives
  • Reads sitemap.xml to discover all listed URLs
  • Follows internal links found in page content, nav, and footer
  • Deduplicates URLs and caps at 50 pages (configurable via MAX_PAGES)

To customize the page limit:

$ MAX_PAGES=100 node server.js https://example.com

Features

Dual Viewport Desktop (1280x800) and mobile (375x812) testing for every page
Live Dashboard Violations stream in real time via SSE with severity and viewport filters
Smart Crawl Sitemap + robots.txt + link discovery finds pages automatically
JSON Export Full report with rule IDs, WCAG tags, selectors, and help links

Requirements

  • Node.js 18+
  • Chromium (auto-installed by Playwright on first run)
